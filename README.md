# 🗂️ Kanban Board

A modern Kanban Board built using HTML, CSS and JavaScript.

## 🚀 Live Demo

https://shashwat14a.github.io/Kanban-Board/

## ✨ Features

- Create, Delete Tasks
- Drag & Drop Support
- Local Storage Support
- Task Counter
- Responsive UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage API
- Drag & Drop API
